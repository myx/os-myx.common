# GetCommonScreenRc ( myx.common os/getCommonScreenRc )

Print default system screenrc path.

Supported OS: Linux, FreeBSD, Darwin.

##  Arguments:

  None.

##  Options:

  None.

##  Usage notes:

  Use --help to print detailed help for this command.

##  Examples:

	# Print default system screenrc path
	`myx.common os/getCommonScreenRc`
