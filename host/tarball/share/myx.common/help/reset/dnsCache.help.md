# DnsCache ( myx.common reset/dnsCache )

Flush DNS resolver cache.

Supported OS: Darwin.
Requires root privileges.

##  Arguments:

  None.

##  Options:

  None.

##  Usage notes:

  Use --help to print detailed help for this command.

##  Examples:

	# Flush DNS resolver cache
	`myx.common reset/dnsCache`
